# Scientific Calculator 🧮

A fully functional **Scientific Calculator** built with HTML, CSS, and JavaScript for college project.

## ✨ Features

- **Basic Operations**: +, -, ×, ÷
- **Scientific Functions**: sin, cos, tan, log, √, x^y, n!, π
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Keyboard Support**: Full keyboard functionality
- **Modern UI**: Glassmorphism design with animations
- **Scientific Mode Toggle**: Switch between basic and scientific modes

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Flexbox, Grid, Animations, Glassmorphism
- **Vanilla JavaScript** - No external dependencies
- **Font Awesome** - Icons
